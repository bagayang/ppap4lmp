Detailed Description

StaDump is an abstract class whose subclasses read a Lammps' dump file.

This class inherits Starter class. This class is an abstract class and has a pure virtual function, StaDump::compute_impl.

◆ timestep

int StaDump::timestep
protected

A timestep of simulation to be read.

In a Lammps' dump file, this value is printed below the line saying ITEM: TIMESTEP. This member is useful to read one timestep from a Lammps' dump file containing multiple timesteps.
