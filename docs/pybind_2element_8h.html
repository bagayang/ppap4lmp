static void pybind_element ( py::module &  m)
static

Function to bind Element class to Python.

Constructor of Element class is hidden from Python. A Python-side function create provides functionality to create an object of Element class with an object of Starter's subclass. Such a function is required to make UpdatePair from shared pointers to the created Element object and Starter object taken as the argument; because constructor cannot call shared_from_this() and therefore cannot make UpdatePair.
