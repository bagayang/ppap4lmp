Detailed Description

StaMolecules sets properties for molecules using atoms forming the molecules.

An object of this class owns an Element object for atoms forming molecules to be computed. Note that if molecular types are required elsewhere, one needs to add type property using an Adder object, such as an AddMap object.

About usage in Python, please see src/pybind/starters/py_sta_molecules.h.

Key of property to be added:

  • id (integer)
  • atom-ids (array of integers)

Key of required property in ext_generator:

  • id (integer)
  • mol (integer)
