Casting Json from C++ to Python.

First, a Json instance is serialized to a string by its dump method. Then the string is loaded by loads attribute of Python's json module and released as a complex of dict and list. This process might take a long time. So, using other containers is preferable (if you can).

Loading Json from Python and assigning it in C++.

First, a JSON variable (complex of dict and list in Python) is dumped as a string by dumps attribute of Python's json module (serialization). Then the string is parsed by parse function of nlohmann/json and converted to a Json instance. This process might take a long time. So, using other containers is preferable (if you can).
