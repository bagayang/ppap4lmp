Detailed Description

The Element class is one of the cores of this program.

This class inherited from the Generator class and EnShThis<Element> (alias of std::enable_shared_from_this<Element>). The Element class is different from the Generator class and its subclasses other than this class by owning the data element (or data entity). The data is stored as a Json (nlohmann::json) instance in the data, and the set of attributes (or property names, keys) of the data is stored as a DataKeys instance in the datakeys. This class can update its own data by calling the update_data method where an editable reference of the data is passed to an Updater instance.

Parameters
constShPtr<Updater> &upd.
Returns
Shared pointer of this Element class.

To update the data, one creates an instance of the Updater class and appends it to an instance of the Element class. UpdatePair in update_chain stored in the external Generator in the Updater instance will be merged to the update_chain of this instance; then the pair consisting of this instance and the instance of the Updater token as the upd will be added to the update_chain of this instance. Please see also the Generator class and Updater class.

Json Element::data
private

Data element (or data entity)

This member is an instance of Json (nlohmann::json), which is a very flexible data container consisting of hierarchically organized pairs of a string key and int/bool/double/string/array value. This can store data of Lammps' Box, Lammps' Atoms, Molecules, Beads, Gaussian's Cube ... and more!!

int Element::n_remain = 0
private

The number of times this class will be used in the future.

This member can be incremented by the increment_remain method and decremented by the decrement_remain method. If this member becomes 0 in the decrement_remain method, the data is cleared to save memory.

A variable used for OpenMP.

This member used to update the data exclusively in a multithread. To prevent the data from being updated from multiple threads, this member must be locked in the update_data.
