Detailed Description

AddChildPositions adds relative positions of a child Element object to a parent Element object.

This class inherits Adder class and overrides Adder::compute_impl.

An object of this class owns a child object as the ext_generator, and is appended to a parent object using its Element::append_updater. An example of child is an Element object containing data for atoms, and an example of parent is an Element object containing data for molecules. Positions of a child is computed relative to a position of a parent and appended to an array belonging to the parent.

The terms child and parent are used because a parent object consists of a child object. In terms of time series, however, the child is created earlier than the parent.

About usage in Python, please see src/pybind/py_adders/add_child_positions.h.

Key of property to be added:

  • [child_name]-xs (array of float)
  • [child_name]-ys (array of float)
  • [child_name]-zs (array of float)

Required key of property:

  • [child_name]-ids (array of integer)
  • xu (float)
  • yu (float)
  • zu (float)

Required key of property in ext_generator:

  • id (integer)
  • xu (float)
  • yu (float)
  • zu (float)
